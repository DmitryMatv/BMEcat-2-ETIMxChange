BMEcat to ETIM xChange Converter 1.0

Upload XML catalog file to convert it to JSON.

The conversion is performed strictly in accordance with the official ETIM mapping documentation, which maps xChange fields to ETIM BMEcat fields where possible. For more information about the ETIM xChange format, visit the ETIM xChange page.

Your data is not stored. Uploaded file is permanently deleted from the server as soon as it is processed, regardless of whether the conversion succeeded or failed.
